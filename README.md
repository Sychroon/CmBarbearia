# Projeto Front-End Web

Este projeto foi desenvolvido como trabalho da disciplina de **Front-End Web**. O objetivo é demonstrar conhecimentos básicos em desenvolvimento de interfaces para web utilizando **HTML, CSS e JavaScript**.

## Funcionalidades

- Estruturação semântica com HTML  
- Estilização com CSS  
- Interatividade básica com JavaScript  
- Layout simples e organizado

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

## Como executar o projeto

Você pode acessar o projeto diretamente pelo GitHub Pages:

https://sychroon.github.io/CmBarbearia/

Ou executar localmente:

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo **index.html** em qualquer navegador.
